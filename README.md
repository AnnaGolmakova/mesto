# Проект: Место

## О проекте
Учебный проект от Яндекс Практикума. Страница с туристическими местами в России, которые было бы интересно посетить.

## Технологии
* HTML5
* CSS3
* Flexbox
* Grid-Layout
* Адаптивная верстка
* БЭМ-методология
* CSS-анимации
* Figma
* JavaScript
* ES6 Modules
* Webpack
* ООП
